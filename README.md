## 🧠 Tech Stack
- Python (Flask)
- Pandas, NumPy
- Scikit-learn (ML model)
- OpenAI API (Generative AI)

## ⚙️ How It Works

2. ML model predicts food demand
3. AI generates optimized menu
4. System shows insights via dashboard
